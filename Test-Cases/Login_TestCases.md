# Login Test Cases

### TC_01: Valid Login
Steps:
1. Open login page
2. Enter valid username
3. Enter valid password
4. Click Login

Expected Result:
User should login successfully

---

### TC_02: Invalid Login
Steps:
1. Enter wrong password
2. Click Login

Expected Result:
Error message should appear

---

### TC_03: Empty Fields
Steps:
1. Keep username and password empty
2. Click Login

Expected Result:
Validation message should appear
