# Checkout Test Scenarios

### Scenario 1: Successful Product Purchase
- User logs in
- User adds product to cart
- User proceeds to checkout
- Payment is successful
- Order confirmation is displayed

---

### Scenario 2: Apply Coupon Code
- User adds product to cart
- User applies valid coupon
- Discount is applied successfully

---

### Scenario 3: Remove Item from Cart
- User adds multiple products
- User removes one product
- Cart updates correctly

---

### Scenario 4: Checkout with Empty Cart
- User goes to checkout without products
- System shows error message
