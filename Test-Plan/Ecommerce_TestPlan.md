# Test Plan - E-commerce Application

---

## 1. Introduction
This document describes the testing strategy for an e-commerce web application.

---

## 2. Scope of Testing
- Login Module
- Product Search
- Add to Cart
- Checkout Process
- Payment Gateway

---

## 3. Types of Testing
- Functional Testing
- Regression Testing
- Smoke Testing
- Sanity Testing
- UI Testing

---

## 4. Tools Used
- JIRA (Bug Tracking)
- Postman (API Testing)
- Chrome DevTools

---

## 5. Test Strategy
- Test cases will be executed for all modules
- Regression testing after every build
- Defects will be logged in JIRA

---

## 6. Exit Criteria
- All critical bugs fixed
- 95% test cases passed
