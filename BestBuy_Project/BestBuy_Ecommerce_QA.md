# 🛒 BestBuy E-commerce QA Project

---

## 👨‍💻 Project Overview
This project simulates end-to-end manual testing of a BestBuy-like e-commerce application.

---

## 🎯 Modules Covered
- User Login & Registration
- Product Search & Filters
- Product Details Page
- Add to Cart
- Checkout Flow
- Payment Gateway
- Order Confirmation

---

## 🧪 Testing Types Performed
- Functional Testing
- Regression Testing
- Smoke Testing
- Sanity Testing
- UI/UX Testing
- Cross Browser Testing

---

## 🐞 Defects Identified
- Cart total mismatch issue
- Payment gateway timeout issue
- Search filter not working correctly
- UI alignment issues on checkout page

---

## 🧾 Deliverables
- 100+ Test Cases Created
- 50+ Bugs Reported in JIRA format
- RTM (Requirements Traceability Matrix)
- Test Scenarios for end-to-end flows

---

## 🔧 Tools Used
- JIRA (Defect Tracking)
- Postman (API Validation)
- Chrome DevTools
- Excel (Test Case Management)

---

## 📊 Outcome
Improved application stability and ensured smooth end-to-end purchase flow for users.
