# E-commerce QA Testing Project

---

## Project Name
E-commerce Web Application Testing (BestBuy / Amazon-like app)

---

## Objective
To ensure the quality of the e-commerce application by performing manual testing on all critical user flows.

---

## Modules Tested
- User Login & Registration
- Product Search & Filtering
- Add to Cart
- Checkout Process
- Payment Gateway
- Order Confirmation

---

## Testing Activities Performed
- Designed and executed 100+ test cases
- Identified and reported 50+ defects in JIRA
- Performed Functional Testing
- Conducted Regression Testing after each release
- Performed UI/UX Testing across browsers
- Verified API responses using Postman

---

## Tools Used
- JIRA (Bug Tracking)
- Postman (API Testing)
- Chrome DevTools
- Excel (Test Case Management)

---

## Outcome
- Improved product stability
- Reduced production defects
- Ensured smooth user experience
